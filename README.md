# Binance Futures Testnet Trading Bot

A simplified Python CLI application to place Market and Limit orders on the Binance Futures Testnet (USDT-M). Built with `python-binance` and `argparse`.

## Prerequisites
- Python 3.8+
- A Binance Futures Testnet Account
